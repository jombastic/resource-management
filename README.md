## Instalation

- Download or clone repository
- Run composer install
- Copy .env.example to .env file and setup app name and database connection
- Run php artisan key:generate to generate application key
- Run php artisan storage:link to generate storage symbolic link to public folder
- Run php artisan migrate to migrate database
- Test my application :)
